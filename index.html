📋 Panduan Operasional Lengkap ▼

📦 1. Data Barang

  • Buka menu Data Barang.
  • Klik tombol Tambah Barang.
  • Masukkan Kode, Nama, dan Satuan.

📥 2. Barang Masuk

  • Pilih kode barang dari daftar.
  • Input jumlah (Qty) & tanggal.
  • Sistem update stok secara otomatis.

📤 3. Barang Keluar

  • Pilih kode barang keluar.
  • Input jumlah dan keterangan.
  • Stok terpotong real-time.

⚠️ 4. Barang Rusak

  • Pilih kode barang rusak.
  • Input jumlah & tanggal kejadian.
  • Data tercatat di riwayat kerusakan.

🔔 5. Monitoring

Stok ≤ 3 unit terpantau otomatis. Jika data belum muncul, tekan Ctrl + R untuk segarkan halaman.

🖨️ 6. Cetak Laporan

Pilih menu Laporan, tentukan rentang tanggal, lalu cetak untuk dokumen fisik/arsip Anda.

Tentang Aplikasi Storeman

Storeman adalah sistem manajemen inventaris cerdas yang dirancang untuk membantu pengelolaan stok barang gudang secara real-time. Aplikasi ini memastikan akurasi data antara barang masuk dan keluar, meminimalisir risiko kehilangan, serta mempercepat proses audit melalui fitur pelaporan otomatis. Dengan antarmuka yang ramah pengguna, Storeman menjadi solusi efisien untuk menjaga ketersediaan barang di gudang Anda.

ℹ️ Apa itu Re-Stok? ▼

Data barang dengan stok ≤ 3 unit.

  • KRITIS: Stok sisa 1 - 3 unit.
  • HABIS: Stok sisa 0 unit.
🚀

Alur Kerja: Setelah Anda menambah atau mengedit data di menu "Data Barang", sistem akan otomatis me-refresh data dan mengarahkan Anda ke tabel ini untuk memverifikasi sisa stok terkini.

⌨️ Jika data tidak muncul, Anda dapat melakukan refresh manual dengan menekan tombol Ctrl + R pada keyboard Anda.
